# 杯中山海 · CREMA ATLAS v2.4

沪上精品咖啡探店手帖 —— **次世代甜点游戏**美术方向的沉浸式单页站点。深可可李子底 × 草莓奶油 / 抹茶薄荷 / 焦糖金三色霓虹，满屏磨砂玻璃 HUD；咖啡杯以**抽象线面 + 全息召唤阵**呈现，杯面跑着真流体拉花模拟；推荐卡是一座**真·3D 空间环绕星环**——滚动与拖拽驱动整环旋转，居中聚焦，底部游戏选择台实时换装；标题与店名以 **CRT 镭射**方式从"屏幕里"蹦出。整站简体中文，零构建、单文件交付。

## v2.5 荐杯司 · 智能推荐（AI 悬浮荐签 / OpenRouter 接入位）

「探」章右下角浮起一枚**悬浮呼吸的「智能推荐」按钮**（磨砂玻璃药丸 × 旋转霓虹描边 × 糖晶闪烁 × 蒸汽杯标；与底部选择台横向相撞时自动升至其上缘），点开手记后详情内是**同一枚按钮**。点击唤出一面**延迟跟随指针的液态玻璃荐签**——霓虹流转描边、扫描线、流光鎏层与角签取景器，内容为「产品推荐」：AI 即兴出杯的饮品名 / 英文名 / 一句荐语 / 两句描述 / 风味签 / 佐点 / 时段 / 出杯 / 预估价，并内置**产品生图框**（当前为程序化拉花示意占位）。荐签认的是**当前选中的那家店**：拖转星环或在手帖内 `‹ ›` 翻店，0.56s 防抖后自动「重酿」为新店内容；近屏幕右/下缘自动翻侧，触屏改为停靠，`prefers-reduced-motion` 下静置。

当前文案与配图为**本地演绎占位**；真实内容规划接入 **OpenRouter**——文案走腾讯混元免费档（`tencent/hunyuan-*:free`），配图走 **nano-banana** 免费档（`google/gemini-2.5-flash-image-preview:free`），提示词模板、挂点契约（`window.AI_RECO_ADAPTER`）、整段参考适配器与安全代理示例见 **`docs/AI-RECO.md`**：注入适配器即自动切换云端（按店缓存 + 换店中断 + 超时回退本地），全程零改动前端其余代码。本模块为**纯增量**，独立坐落于 `index.html` 尾部（`#aiRecoStyle` / `#aiRecoApp` 两段，带界碑注释），未触碰既有十二系统的任何一行。

## v2.4.1 液态玻璃悬浮舱

详情主图升级为三倍体量的「液态玻璃悬浮舱」：液玻厚板托底（毛玻璃折射 + 全息描边），照片内叠 SVG 液态折射层与流光鎏层；舱体 preserve-3d，鼠标移动时英文名、编号徽、星级片、商圈签四枚文字层在 56/74/84/98px 深度上真实错位飞出；悬停整舱被一层黏连金属球式的半透明奶流体缓缓漫过（与碎裂玻璃遮罩并存），离开即回缩。原有取景器特效（追光、速度色散、碎镜、电影颗粒、跑马字条、快门栏）全部保留。

## v2.4 漩涡盘与净杯

按需求**拆除杯口以上全部渲染**（奶液面、注奶流、蒸汽、奶滴与拉花导演），白色团块自此不可能存在；杯保留抽象线面 + 召唤阵 + 糖晶物理，「再拉一杯」按钮改为「撒把糖晶」。移植参考工程最深背景层 **buildVortex 拉花漩涡盘**：九千粒子的巨型差速旋转盘（内圈快外圈慢）斜置于主场景深处——滚轮冲量为它加速并加剧起伏，页面向下推进时整盘膨胀、抬升、渐散；鼠标是"搅拌勺"：指针速度与按压决定力度，粒子沿径向被推开、沿切向被卷绕。配色换为甜点大理石谱（深可可李子→草莓→奶油→焦糖金，7% 抹茶碎点），漩涡亦会倒映进底部海面。

## v2.3 山海化（v2.3.1 修正：海面顶点定位回到视口最底部；液面弃白改奶咖大理石全域纹）

背景加入**三层视差远山**（山脊随鼠标与滚动错动、脊线各镀粉/金/薄荷微光）与**双层漂浮云雾**（时间驱动，另有三团 DOM 云缓慢横渡）；页面底部升起一片**实时倒影之海**——整幕 3D 场景（杯、粒子、远山）被翻转映入水中，三重波形叠加全息浪尖与随机闪点，指针掠过水域会激起涟漪，尾声大字亦倒映水面。杯面弃用刺眼白色光球，改为**卡布奇诺白褐相间的大理石流纹**（噪声流场采样 + 褐色纹络 + 收敛高光，注奶时播撒褐色种子供纹路生长）。详情页：店铺主图放大近一倍并叠**电影滤镜**（分级提档 + 暗角 + 动态胶片颗粒），悬停主图与画廊首图会浮现**半透明碎裂玻璃遮罩**——三角碎片随指针大面积推挤、旋转、迸出糖屑粒子；详情内全部文字与框体获得整页 **VR 视差**（十余层深度 + 标题/主图/雷达附加倾斜），活背景亦补上双层山脊、雾带与底部微浪倒光。

## v2.2 供体移植（源自 activetheory 镜像工程）

从随附的生产级 shader 包（`compiled.vs`，250 个模块）中考古并移植了四套语法，全部换装为本站甜点色板：详情页新增**活体背景**（呼吸角光 × 三道旋弧 × 径向扫描场 × 右上神光 × 闪烁糖尘 × 胶片颗粒，随店铺切换主音色、随滚动视差）；所有影像框升级为**取景器**——鼠标追光辉斑、指针速度驱动的红青色散、Ken Burns 微游移、四角托架、顶部英文跑马字条与底部快门参数栏；液幕转场在遮罩边缘并入 WorkComposite 的**全息扩散环与放射条纹**，并伴随一记 DOM 冲击波；详情内容加厚：左缘章节栏（引/数/志/影随滚动点亮）、冲煮参数六格、到访手记时间线、三枚甜点勋章，且标题/小节标随滚动速度产生 RGB 动势。

## 运行

双击 `index.html` 即可（推荐 Chrome / Edge / Safari 最新版）。想让本地 `data/cafes.json` 生效，起个静态服务：

```bash
cd crema-atlas
python3 -m http.server 8080   # 打开 http://localhost:8080
```

首载会经过完整的「甜点引擎启动」加载演出（启动日志 / 漩涡星云 / 进度圆环 / 虹膜开场）。three.js 走 cdnjs；字体、第三方图片、WebGL 任何一环缺席都会自动降级，不会白屏。

## 交互速览

滚轮/触摸驱动虚拟滚动，四章「序 · 沏 · 探 · 记」推进；HERO 右侧五块玻璃仪表（风味雷达 / 奶温表盘 / 烘焙度 / 香气矩阵 / 手账印）带强视差与鼠标倾斜，像悬在你面前的驾驶舱。进入「探」章，页面钉住、七张卡自天而降旋成星环：**左右拖拽**或滚轮旋转星环，居中卡点亮全息描边，底部选择台同步编号、镭射店名、星级与人均，`‹ ›` 步进、「展开手记」或点击焦点卡开卷；筛选商圈会让旧卡飞散、新环重组。悬停杯口可直接搅拌液面（光标变「搅」），「再拉一杯」重播整套罗塞塔注奶；卡片图片做了甜点系色彩分级，悬停碎镜位移 + 红青色散 + 电流扫描。

## 自定义与性能

换主题只动三处：`:root` 设计令牌、`Latte` 注入路径、`KF` 场景关键帧；方法论见同批 `immersive-webgl-frontend` 技能（含空间环绕画廊 / CRT 镭射文字 / 玻璃 HUD 三个新配方）。内置 FPS 自动降级（粒子 → 压力迭代 → DPR），尊重 `prefers-reduced-motion`，无 WebGL 时切换纸上静读模式。
